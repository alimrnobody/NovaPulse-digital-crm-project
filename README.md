# NovaPulse Client Onboarding Portal

A full-stack automation system built for a fictional digital marketing agency **NovaPulse Digital**, combining a modern React client portal with powerful CRM automation using GoHighLevel (GHL) and n8n workflows.

---

## 📌 Project Overview

This project automates the entire client onboarding journey — from signup to project kickoff — eliminating manual processes and creating a seamless client experience.

👉 Built as an advanced automation system integrating:
- React Client Portal
- GoHighLevel CRM
- n8n Automation Workflows
- Google Sheets Logging

---

## 🧠 Problem Solved

Agencies typically onboard clients manually using emails, forms, and follow-ups.

❌ Problems:
- Slow onboarding (3–5 days)
- Manual follow-ups
- Data scattered across tools
- No real-time tracking

✅ Solution:
A **fully automated onboarding system** with:
- Self-service client portal
- Automated CRM pipeline movement
- Real-time workflow automation
- Centralized logging & tracking
